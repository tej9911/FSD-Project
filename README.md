# 🎯 Event Management System

A full-stack web application designed to simplify event creation, management, and user registration. This system provides a centralized platform where users can explore events, register for them, and track participation, while administrators can efficiently manage events and users.

---

## 🚀 Features

### 👤 User Features
- User registration and login authentication
- Browse and explore available events
- View detailed event information (date, venue, description)
- Register for events بسهولة
- View registered events in user dashboard

### 🛠️ Admin Features
- Create, update, and delete events
- Manage users and event registrations
- Monitor participation and event data

### 🔐 Security Features
- Secure authentication system
- Session-based login management
- Protected routes for authorized access
